# 🌦️ Weather Data - Data Lake Project 
This project simulates a *Data Lakehouse* for ingestion, processing and analysis of weather data using: 

- Python
- OpenWeather API
- Parquet
- Bronze / Silver / Gold layers
- SQL for aggregations

## 📌 Main Objective
Demosntrate practical habilities in Data Engineering such as: 

- API ingestion
- Data cleaning
- Partitioning and Storing
- Layer modeling
- Analytic aggregations
